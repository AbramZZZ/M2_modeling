# Моделирование М2

Реализованы классы для банка и их совокупности. Восстановить М2 по активам и пассивам легко.

А вот обратно уже труднее. Ведь переменных у нас будет больше, чем условий на них. Сделал упрощенную модель(decompose_m2) которая возвращает активы и пассивы, используя какие-то константы. Планирую добавить поиск констант(которые минимизируют какую-нибудь функцию, например, сумму среднеквадратичных отклонений модели от реальных данных). Пока модель работает так:

```
M0 = cash_outside
M1 = M0 + deposits_checking
M2 = M1 + deposits_savings + deposits_time
резервы = reserve_ratio * deposits_checking
securities = securities_share * total_deposits
```

Доли банков определяются от их "весов"
